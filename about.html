About Run Houston

A community race guide for the Bayou City.

What this is

Run Houston is a free, community-focused directory of road, trail, and track races happening in and around Houston, Texas. It also lists local running clubs and publishes first-person race reports from runners who actually showed up at the starting line.

How it's built

This site is intentionally simple: plain HTML, CSS, and JavaScript — no framework, no build step, and no server. Every page loads its data from flat JSON files under /data/ and queries them in the browser with AlaSQL. The whole thing is hosted on GitHub Pages.

How the data stays fresh

There is no admin login, no CMS, and no database. Race calendars, club listings, and report drafts are kept current by scheduled GitHub Actions that ask the Anthropic API to do web research on Houston-area races and propose updates to the JSON files. A human reviews the changes before they ship.

That means data on this site is best-effort. If something looks wrong — a bad date, a dead link, a missing race — please open an issue on the GitHub repo.
